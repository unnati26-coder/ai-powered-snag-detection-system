## Overview
An AI-powered construction snag and defect detection system built using computer vision and automated inspection workflows.

The system detects structural defects such as cracks, dampness, and corrosion from construction site images while generating severity analysis, inspection reports, annotated outputs, and automated maintenance recommendations.

---

## Features
- AI-powered snag detection
- Crack, dampness, and corrosion detection
- Automated defect severity analysis
- Bounding box visualization
- Inspection report generation
- Automated email draft generation
- Feedback learning workflow
- Defect analytics pipeline
- Image processing and annotation
- Construction inspection automation

---

## Tech Stack
- Python
- OpenCV
- YOLOv8
- Computer Vision
- NumPy
- JSON
- Automated Inspection Pipeline

---

## System Workflow
1. Upload construction defect image
2. Detect structural defects using vision model
3. Analyze severity level automatically
4. Generate inspection report
5. Create automated maintenance recommendation
6. Draw annotated defect boundaries
7. Store user feedback for learning improvements
